User Flow

Automated Settlement Cycle

How autonomous agents consume and pay for data in four simple steps.

Step 01

Resource Request

The agent makes a standard GET request to any gated endpoint.

Step 02

402 Challenge

Server returns 402 with PAYMENT-REQUIRED header containing amount/wallet.

Step 03

Native Settlement

Agent signs and broadcasts USDC on Solana or Base.

Step 04

Instant Unlock

Agent resubmits with TX hash. Server verifies and delivers data.

Prompt Examples

These examples are designed for the public remote MCP discovery server at /mcp/server/. They demonstrate the free discovery layer that routes users into the paid HTTP API.

1. Discover supported symbols

Use the Blocksize discovery connector only.
Search for instruments related to BTC, SOL, and EURUSD.
Then tell me which exact symbols and services are supported.

2. Price and payment planning

Using the Blocksize discovery connector, inspect the pricing tiers.
Then explain the cheapest way to pull BTCUSD, ETHUSD, EURUSD, and AAPL data,
including whether credits or direct x402 settlement are better.

3. Integration handoff

Use Blocksize search and fetch to find the remote MCP quickstart and the pricing guide.
Summarize the exact next steps for a developer who wants to:
1. add the discovery MCP to Cursor
2. call the paid HTTP API for live data
3. top up credits for an agent wallet

4. Catalog fetch with citations

Use the Blocksize connector's search tool to find docs about privacy and support.
Fetch the most relevant documents and summarize the logging policy and support channel,
with links back to the original docs.

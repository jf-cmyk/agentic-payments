# First Batch Account Briefs

## 1. Dune

Domain: `dune.com`

Priority: A

Best route: Hunter Domain Search for `dune.com`; look for API, data platform, partnerships, ecosystem, community, developer relations, or product contacts.

Why now: Dune already serves programmatic onchain analytics, APIs, datasets, dashboards, and agent-readable workflows. The Blocksize wedge maps naturally to one paid premium query that agents can buy on demand.

Personalized opener:

> Dune is one of the clearest examples of onchain data becoming a programmable surface for apps and agents.

Primary ask:

> Would you review a 5-minute demo of an agent paying for one financial-data response over HTTP 402/x402 and tell us whether this belongs in the onchain data stack?

Discovery angle:

- Which Dune query or data slice would be valuable enough for per-call payment?
- Would x402 be useful as a billing option for machine clients?
- Would Dune prefer to distribute, consume, or ignore this primitive?

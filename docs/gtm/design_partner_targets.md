## First outreach angle

Subject options:

- "Agent pays for institutional market data over HTTP 402"
- "Looking for design partners for x402/MCP financial data"
- "Can your agents buy market data without accounts or API keys?"

Core ask:

Ask for a 20-minute design partner conversation, not a sale.

Core validation questions:

- What would your agents pay for today if checkout/API keys were not a blocker?
- Where do your current agent workflows need financial data?
- Do you prefer per-call payments, prepaid credits, or subscription bundles?
- What proof would make you trust an agent-paid data call?
- Which protocol or wallet stack are you already using?

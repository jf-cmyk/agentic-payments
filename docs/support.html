Before You Contact Us

  • Confirm the remote MCP URL is https://agentic-payments-production.up.railway.app/mcp/server/.
  • Check the health endpoint for current status.
  • Verify installation steps in the quickstart.
  • Review the OpenAPI docs if your issue is related to paid HTTP access.

What to Include

  • The client you are using, such as ChatGPT Developer Mode, Cursor, Claude, Smithery, or a custom runtime.
  • The MCP URL or HTTP endpoint you attempted to call.
  • The tool name or endpoint name that failed.
  • The approximate time of failure and any relevant error text.
  • Whether the issue affects the free discovery server or the paid HTTP API.
